## Excluded pending verified material

T-shirts, sweatshirts, knitwear, outerwear, trousers, accessories, dresses, labels/patches and separate design-process galleries do not have enough media whose garment classification can be confirmed from the current record. To publish these as focused pages, provide approved, high-resolution exports grouped by project and garment type, plus any relevant process, label, patch or production imagery. For Flasheros, provide approved original photography exports if a longer private-project sequence is desired.
